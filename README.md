- Input handling  
- Data types

**Logic Used:**

- for loops  
- Iteration

**Logic Used:**

### 🔹 Day 6 – Smart Transaction Risk Detector 💳
**Concepts Used:**
- Lists  
- Dictionary  
- Tuple  
- List comprehension  
- Conditional logic  

**Logic Used:**  
I categorized transactions into normal, large, high-risk, and invalid using conditions. Then I filtered valid transactions using list comprehension to calculate total value and count. After that, I checked three conditions: frequent transactions, large spending, and high-risk occurrences. I counted how many conditions were true and used that to determine the final risk level.
